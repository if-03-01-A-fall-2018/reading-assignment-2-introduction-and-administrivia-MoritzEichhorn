Web Server

What is it?
  • is a program that servers the files that form Web pages to users via HTTP
What does it?
  • stores, processes and delivers web pages to clients
Storage
  • The resource is typically a real file on the server's secondary storage, but it depends on how it is implemented

Web Client

What is it?
  • It is a software application for accessing information on the WWW
What does it?
  • fetches information rescources and displays them on a user's device

Font
